# Dziennik

> Dziennik do zapisywania dobrych wiadomości.

Prosta aplikacja webowa do prowadzenia codziennego dziennika.

- maks. **5 notatek dziennie** (po 500 znaków),
- **kalendarz miesięczny** z oznaczeniem dni z wpisami,
- **edycja 7 dni wstecz**, starsze dni tylko do odczytu,
- **codzienne przypomnienie** o wybranej godzinie (Notification API),
- eksport / import danych (JSON),
- dane lokalnie w `localStorage` przeglądarki – bez backendu.

## Uruchomienie

Otwórz [`index.html`](index.html) w przeglądarce (dwuklik). Tyle.
